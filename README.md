# Formação Acelerada em Programação (FAP)

## Sobre o Projeto

Este repositório reúne atividades, projetos, desafios e materiais desenvolvidos durante a Formação Acelerada em Programação (FAP), uma iniciativa voltada para capacitação prática em tecnologia e desenvolvimento de software.

A formação utiliza metodologias ativas de aprendizagem, com foco em resolução de problemas reais, trabalho em equipe e desenvolvimento de soluções aplicadas ao mercado.

## Objetivos

- Desenvolver competências técnicas na área de tecnologia.
- Aplicar conceitos em projetos práticos.
- Trabalhar com metodologias ágeis e colaborativas.
- Construir portfólio profissional.
- Preparar para oportunidades no mercado de trabalho.

## Estrutura da Formação

- **Carga Horária Total:** 215 horas
- **Horas Presenciais:** 159 horas
- **Horas Online (AVA):** 56 horas

### Metodologias Utilizadas

- PBL (Problem Based Learning)
- Sala Invertida
- Projeto Integrador
- Demoday

## Tecnologias

As tecnologias utilizadas poderão variar ao longo da formação, incluindo:

- Git e GitHub
- Linux
- Docker
- DevOps
- Cloud Computing
- Automação
- Integração Contínua (CI/CD)

## Projetos Desenvolvidos

Durante a formação serão adicionados:

- Projetos individuais
